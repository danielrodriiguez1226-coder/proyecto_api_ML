# Hand Keypoint Detection with YOLO

Proyecto de detección de puntos clave de la mano usando **YOLOv8 Pose**.

## Descripción
Este proyecto permite detectar manos y contar los dedos levantados en imágenes y videos.

## Archivos principales
- `detect_photo.py` : Detecta manos en imágenes.
- `detect_video.py` : Detecta manos en videos.
- `count_fingers.py` : Función para contar dedos levantados.
- `weights/` : Pesos entrenados del modelo (best.pt, last.pt, etc.)
- `dataset.yaml` : Configuración del dataset para entrenamiento.
